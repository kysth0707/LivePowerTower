## live.py
### 사용법
- 드래그 : 확대, 다른 마우스 버튼 클릭 시 취소
- 휠 : 이동, 다른 마우스 버튼 클릭 시 취소
- 축소 : 우클릭, 2배 축소
-----------
- 아직 확대 부분의 정확도가 떨어집니다
- 임시 렌더가 끝나면 터미널 창에 x y eps가 표시됩니다
- 표시된 좌표를 highquality.py 코드 안에 붙여넣고 실행하시거나, 자신만의 세팅 값에 맞춰 렌더하시면 됩니다
